HOOBS software repository.


## Installing
First add the HOOBS repository to your sources.

```sh
wget -qO- https://dl.hoobs.org/stable | sudo -E bash -
```


Now you can install hoobsd and the HOOBS CLI.

```sh
sudo apt install -y hoobsd hoobs-cli
```
